# Duothan 3.0



## How to use

### 1\. Clone this Repository

```bash
https://github.com/IEEE-Student-Branch-NSBM/Duothan3.0-Website.git
```

### 2\. Navigate to the directory

```
cd Duothan3.0-Website
```

### 3\. Install dependencies

```
yarn install
# or
npm install
```

### 5\. Run Dev Server

```
yarn dev
# or
npm run dev
```
